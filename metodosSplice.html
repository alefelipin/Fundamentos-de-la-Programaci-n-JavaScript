JavaScript Arrays:

.slice() vs .splice()

Cuándo usarlos?

.slice(inicio, fin): cuando necesitas extraer una parte de un array. El array original se mantiene

.splice(inicio, cantidad, nuevos items): cuando necesitas modificar el contenido (eliminar, reemplazar o añadir elementos). El array original se modifica. Devuelve un array que contiene los elementos eliminados (si hay).
